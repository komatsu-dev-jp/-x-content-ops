# Banned Patterns

## 禁止表現

- 絶対勝てる
- 勝率爆上げ
- 誰でも稼げる
- パチンコで生活できる
- 必勝
- 攻略法
- 爆益
- 最強
- 革命
- 人生変わる
- 今すぐ登録
- これを使わない人は損
- 勝ち組になれる
- 期待値だけで勝てる

## 避けたい投稿パターン

### 1. 勝ち保証型

Bad:

```text
PachiTrackerを使えば勝率が上がります。
```

Good:

```text
PachiTrackerは、判断を後から見直すための記録アプリです。
```

### 2. 機能説明だけ型

Bad:

```text
回転率分析、収支管理、期待値計算を実装しました。
```

Good:

```text
「回ってる気がする」を、あとで数字で見直せるようにしました。
```

### 3. 煽り型

Bad:

```text
まだ感覚で打ってる人は危険です。
```

Good:

```text
感覚で打つと、あとで判断理由が変わることがあります。
```

### 4. 内部ロジック漏洩型

Bad:

```text
ベイズ推定でこの閾値を超えたら撤退判定します。
```

Good:

```text
判断に迷う場面を、後から見直せるようにしています。
```
